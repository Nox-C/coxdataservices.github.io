# Cox Data Services Website

This is the official website for Cox Data Services.

## Features

*   **Home Page:** Provides an overview of the company and its services.
*   **Products Page:** Displays the products and pricing offered by the company.
*   **Consciousness Page:** An interactive experiment in artificial intelligence.
*   **Contact Page:** A form to contact the company.
*   **Hookie Page:** A fun, interactive experience that simulates a data scraper and provides a data-driven fortune.

## Technologies Used

*   HTML5
*   CSS3
*   JavaScript
*   [three.js](https://threejs.org/)
*   [PayPal REST API](https://developer.paypal.com/docs/api/overview/)

## How to Deploy

This website is deployed using GitHub Pages. To deploy the website, simply push the changes to the `master` branch of the GitHub repository.
